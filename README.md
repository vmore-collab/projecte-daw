# 📦 StudentShare - Gestió de material

Aplicació web desenvolupada com a projecte de 2n de DAW. La plataforma permet als estudiants compartir i gestionar el préstec de material (calculadores, llibres, teclats).

## 👥 Equip
- **Product Owner:** vmore-collab
- **Scrum Master:** vmore-collab
- **Developers:** vmore-collab

## 🛠️ Tecnologies
- **Frontend:** React + Tailwind CSS
- **Backend:** Node.js + Express
- **Base de dades:** MongoDB
- **Control de versions:** Git + GitHub

## 🔄 Metodologia
Aquest projecte es desenvolupa utilitzant SCRUM amb sprints d'1 setmana.

## ⚙️ Instal·lació
```bash
git clone [https://github.com/vmore-collab/projecte-daw.git](https://github.com/vmore-collab/projecte-daw.git)
npm install
